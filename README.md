# 湘M顺风车发布平台

一个专为湘M地区设计的顺风车信息发布和查找平台。

## 项目特色

- 🚗 顺风车信息发布
- 🔍 智能搜索功能
- 💾 本地存储数据持久化
- 📱 响应式设计
- 🎨 橙色主题配色

## 功能特性

### 发布行程
- 填写出发地、目的地、时间等信息
- 设置座位数量和费用
- 添加联系方式和备注

### 查找行程
- 根据出发地或目的地搜索
- 查看详细行程信息
- 获取联系方式

### 我的行程
- 查看已发布的行程
- 删除不再需要的行程

## 部署方式

### 方法一：GitHub Pages（推荐）
1. 将项目上传到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择 main 分支作为源

### 方法二：Netlify
1. 将项目上传到 GitHub/GitLab
2. 在 Netlify 中连接仓库
3. 自动部署

### 方法三：直接访问
- 直接在浏览器中打开 `index.html` 文件

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage 数据存储
- Font Awesome 图标

## 使用说明

1. 发布行程：填写完整信息后点击"发布行程"
2. 查找行程：在搜索框中输入关键词
3. 管理行程：在"我的行程"中查看和删除

## 数据存储

所有数据保存在浏览器的 LocalStorage 中，确保：
- 页面刷新后数据不丢失
- 同一浏览器内数据持续保存
- 用户可以自主管理数据
